Python默认参数值

发表于 2017-12-07 | 分类于 Python

对于python中的任意变量，id函数可以显示该变量在内存中的地址。
当给函数传递变量时，可以认为传递的是该变量的地址值。

对于小整数和字符串类型，值相同，地址值是相同的
对于list来说，即使两个list类型变量值相同，但地址值是不同的

对于有默认参数的函数，在没有传入参数时，函数会使用默认参数，该默认参数的地址在定义函数时已经确定,且不会更改

进一步说明：在没有给func函数传递参数时，func函数使用的是args的值，假设args变量的地址是43891008，不管之后怎么调用该函数，只要不传参数，使用默认的args参数，func函数使用的仍然是地址为43891008的args变量

每调用一次func函数，都会在args列表末尾添加1
当传递空列表时，传递的是该空列表的地址值。
再次调用func函数，使用的是args变量，该变量在上次运算中为[1, 1, 1]
